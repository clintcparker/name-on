# Feature Ideas
1. Customization Options

- Format templates: Let users choose patterns like adjective-noun, noun-adjective-number, adjective-adjective-noun, etc.
- Separator options: dash, underscore, camelCase, or no separator
- Number length/range: control digits (0-999 vs 0000-9999)
- Word length limits: short names only, or allow longer words

2. Theme Packs / Word Lists

- Tech/Dev: quantum-stack-42, agile-pipeline-88
- Nature: misty-forest-17, rolling-thunder-33
- Food: spicy-mango-12, golden-waffle-55
- Fantasy/Gaming: shadow-knight-99, mystic-dragon-07
- Custom word lists: let users upload or paste their own adjective/noun lists

3. Bulk Generation

- Generate 5, 10, 50 names at once
- Export to clipboard, JSON, CSV, or text file
- "Lock" favorites and regenerate the rest

4. Uniqueness & Validation

- Collision checker: paste existing names to ensure new ones don't match
- Domain availability check: quick link to see if clever-otter-123.com is available
- GitHub repo name check: is it taken?
- "Profanity filter": avoid unfortunate word combinations

5. History & Favorites

- LocalStorage history of generated names
- Star/favorite names for later
- "Copy all favorites" button

6. API / Integration

- Simple REST/JSON API endpoint (could be serverless)
- CLI tool (you mentioned this in the roadmap)
- Browser extension for quick generation
- Slack/Discord bot integration

7. Shareable Links

- Permalink to a specific generated name: name-on.site/s/clever-otter-123
- QR code generation for sharing

8. Fun Extras

- Name pronunciation: audio or phonetic spelling
- Name meaning: generate a silly "etymology" for the combo
- Daily name: a consistent name-of-the-day everyone sees
- Name roulette: spin animation before revealing
